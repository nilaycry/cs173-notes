Connectivity turns paths into a global property of a graph. Trees are the graphs where connectivity exists without any cyclic redundancy.

## Connected Graphs

> **Definition.** A graph $G$ is connected if every pair of vertices in $G$ is joined by a path. Formally,
>
> $$
> (\forall x,y\in V(G))(\exists k\in\mathbb{N})(\exists p:k+1\to V(G))
> \bigl(p\text{ is a path}\wedge p(0)=x\wedge p(k)=y\bigr).
> $$

If $x=y$, the path can have length $0$. If $x\neq y$, connectedness says there is a genuine route through edges from $x$ to $y$.

Because every walk between two vertices can be shortened to a path, it is equivalent to define connectedness using walks. The path version is cleaner because it removes irrelevant repetitions.

## Disconnected Graphs

> **Definition.** A graph $G$ is disconnected if it is not connected.

Equivalently,

$$
(\exists x,y\in V(G))(\text{there is no path in }G\text{ from }x\text{ to }y).
$$

When proving that a graph is disconnected, it is enough to produce two vertices that cannot be joined by a path.

## Components

> **Definition.** A connected component of $G$ is a maximal connected subgraph of $G$.

Maximal means that the component cannot be enlarged by adding another vertex of $G$ and still remain connected.

If two vertices lie in different components, there is no path between them in $G$. If an edge joined vertices in two different components, those components would actually be part of the same connected component.

## Acyclic Graphs and Forests

> **Definition.** A graph is acyclic if it contains no cycles.

> **Definition.** A forest is an acyclic graph.

A forest may be disconnected. Each connected component of a forest is a tree.

## Trees

> **Definition.** A tree is a connected acyclic graph.

Thus

$$
T\text{ is a tree}
\quad\Longleftrightarrow\quad
T\text{ is connected and }T\text{ has no cycles}.
$$

Both conditions matter.

- A connected graph with a cycle is not a tree.
- An acyclic graph that is disconnected is a forest, not a tree.

## Leaves and Internal Vertices

> **Definition.** Let $T$ be a tree. A vertex $v\in V(T)$ is a leaf if
>
> $$
> \deg_T(v)=1.
> $$

> **Definition.** A vertex of a tree that is not a leaf is called an internal vertex.

In a one-vertex tree, the only vertex has degree $0$, so it is not a leaf under this definition.

## Standard Tree Facts

The following theorems drive most first graph-theory proofs:

- Every tree has a unique path between any two vertices.
- Every tree with at least two vertices has a leaf.
- Every nonempty tree satisfies $|V(T)|=|E(T)|+1$.
- Removing a leaf from a tree produces a smaller tree.
- Every connected graph contains a spanning tree.

## Core Summary

Connected graphs have paths between all pairs of vertices. Forests have no cycles but may be disconnected. Trees are exactly the connected forests. Most tree proofs use both halves of the definition: connectedness supplies paths, and acyclicity rules out alternate routes.

Graph theory begins by replacing extra context with a finite set of objects and a set of pairwise connections. The definitions are set-theoretic because later proofs count vertices, edges, neighbors, and incident edges.

## Simple Graphs

> **Definition.** A graph $G$ consists of a finite vertex set $V(G)$ and an edge set $E(G)$ such that
>
> $$
> E(G) \subseteq \{\, e \subseteq V(G) \mid |e| = 2 \,\}.
> $$

Each edge is a two-element subset of $V(G)$. If $v,w \in V(G)$ with $v \neq w$, then $\{v,w\}$ is the edge connecting $v$ and $w$.

This definition gives the kind of graph used in this unit: finite, simple, and undirected.

- No self-edge is possible, because an edge must contain two distinct vertices.
- No multiple edges are possible, because $E(G)$ is a set.
- No direction is present, because $\{v,w\}=\{w,v\}$.

## Vertices, Edges, Order, and Size

> **Definition.** The elements of $V(G)$ are the vertices of $G$. The elements of $E(G)$ are the edges of $G$.

> **Definition.** The order of $G$ is $|V(G)|$. The size of $G$ is $|E(G)|$.

Order counts vertices. Size counts edges. These two cardinalities are different pieces of data, and many graph proofs are about how one constrains the other.

## Neighborhoods

> **Definition.** Let $G$ be a graph and let $v\in V(G)$. The neighborhood of $v$ in $G$ is
>
> $$
> N_G(v) \coloneqq \{\, w \in V(G) \mid \{v,w\} \in E(G) \,\}.
> $$

The neighborhood is a set of vertices. If $w\in N_G(v)$, then $w$ is adjacent to $v$.

In a simple graph, $v\notin N_G(v)$ for every $v\in V(G)$, because $\{v,v\}$ is not a two-element set.

## Incidence Sets

> **Definition.** Let $G$ be a graph and let $v\in V(G)$. The incidence set of $v$ in $G$ is
>
> $$
> I_G(v) \coloneqq \{\, e \in E(G) \mid v \in e \,\}.
> $$

The incidence set is a set of edges. This is not the same kind of object as the neighborhood:

- $N_G(v)$ contains vertices.
- $I_G(v)$ contains edges.

In a simple graph, these two sets have the same cardinality. Each neighbor $w\in N_G(v)$ corresponds to the unique incident edge $\{v,w\}\in I_G(v)$, and each incident edge $\{v,w\}\in I_G(v)$ corresponds to the unique neighbor $w$.

## Degree

> **Definition.** The degree function of $G$ is
>
> $$
> \deg_G:V(G)\to \mathbb{N}
> $$
>
> defined by
>
> $$
> \deg_G(v) \coloneqq |I_G(v)|.
> $$

Degree counts incident edges. In a simple graph, it also counts neighbors:

$$
\deg_G(v)=|I_G(v)|=|N_G(v)|.
$$

This equality uses simplicity. It depends on the fact that there is at most one edge between two vertices.

## Isolated and Universal Vertices

> **Definition.** A vertex $v\in V(G)$ is isolated if $\deg_G(v)=0$.

An isolated vertex has no incident edges and no neighbors.

> **Definition.** If $|V(G)|=n$, a vertex $v\in V(G)$ is universal if $\deg_G(v)=n-1$.

A universal vertex is adjacent to every other vertex in the graph. The value $n-1$ is the largest possible degree because a vertex cannot be adjacent to itself.

## Degree Bounds

> **Proposition.** If $G$ is a graph with $|V(G)|=n$ and $v\in V(G)$, then
>
> $$
> 0\leq \deg_G(v)\leq n-1.
> $$

**Proof.** Since $\deg_G(v)=|N_G(v)|$, degree is nonnegative.

Also,

$$
N_G(v)\subseteq V(G)\setminus\{v\}.
$$

Therefore

$$
\deg_G(v)=|N_G(v)|\leq |V(G)\setminus\{v\}|=n-1.
$$

This proves the bound.

## The Impossible Pair

> **Proposition.** Let $G$ be a graph with $|V(G)|=n\geq 2$. If $G$ has an isolated vertex, then $G$ has no universal vertex.

**Proof.** Let $x\in V(G)$ be isolated. Then $N_G(x)=\varnothing$.

Suppose, for contradiction, that $y\in V(G)$ is universal. Since $n\geq 2$, either $y=x$ or $y\neq x$.

If $y=x$, then $y$ is both isolated and universal, so $\deg_G(y)=0$ and $\deg_G(y)=n-1$. Since $n\geq 2$, this is impossible.

If $y\neq x$, then universality of $y$ gives $\{x,y\}\in E(G)$. That means $x$ has an incident edge, contradicting that $x$ is isolated.

Therefore a graph cannot have both a degree-$0$ vertex and a degree-$(n-1)$ vertex.

## Common Mistakes

- Treating an edge as an ordered pair instead of a two-element set.
- Confusing $N_G(v)$, a set of vertices, with $I_G(v)$, a set of edges.
- Forgetting that $\deg_G(v)$ is defined as a cardinality.
- Assuming degree values can always use all of $0,1,\dots,n-1$ at once.

## Core Summary

A graph in this unit is a finite simple undirected graph. Edges are two-element subsets of the vertex set. Neighborhoods contain adjacent vertices, incidence sets contain incident edges, and degree is the size of the incidence set. In a graph with $n$ vertices, every degree lies between $0$ and $n-1$, but the extreme values $0$ and $n-1$ cannot both occur when $n\geq 2$.

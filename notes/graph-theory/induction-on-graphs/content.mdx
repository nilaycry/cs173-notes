Graph induction is ordinary induction applied to a number attached to a graph. The hard part is not the induction template. The hard part is constructing the smaller graph so that the hypotheses still apply.

## Choosing An Induction Parameter

The most common parameters are:

- $|V(G)|$, the order of the graph
- $|E(G)|$, the size of the graph

The theorem determines the parameter.

- The degree-sum theorem works well by induction on $|V(G)|$, because deleting a vertex gives a smaller graph and changes degrees in a controlled way.
- The spanning-tree theorem works well by induction on $|E(G)|$, because deleting an edge keeps the vertex set fixed.
- The tree equation $|V(T)|=|E(T)|+1$ works well by induction on $|V(T)|$, because deleting a leaf removes exactly one vertex and one edge.

## Standard Form

> **Induction Form.** If the parameter is $|V(G)|$, write the theorem as
>
> $$
> (\forall n\in\mathbb{N})(\forall G)
> \bigl((G\text{ is in the class under discussion}\wedge |V(G)|=n)\Rightarrow P(G)\bigr).
> $$

If the parameter is $|E(G)|$, write

$$
(\forall n\in\mathbb{N})(\forall G)
\bigl((G\text{ is in the class under discussion}\wedge |E(G)|=n)\Rightarrow P(G)\bigr).
$$

The statement must quantify over all graphs with the given parameter value. Proving the result only for graphs built by a particular construction is not enough.

## Standard Pattern

1. Choose the parameter.
2. Write the theorem in quantified form.
3. Prove the smallest relevant case.
4. Assume the statement for parameter value $k$.
5. Let $G$ be an arbitrary graph with parameter value $k+1$.
6. Construct a smaller graph $H$.
7. Prove that $H$ still satisfies the hypotheses.
8. Apply the inductive hypothesis to $H$.
9. Transfer the conclusion from $H$ back to $G$.

## What Must Be Checked

When a proof modifies a graph, the writeup must say what survives.

- If a vertex is deleted, check which edges were deleted and how degrees changed.
- If an edge is deleted, check whether connectedness was preserved.
- If a leaf is deleted from a tree, check that the remaining graph is still connected and acyclic.
- If a cycle edge is deleted, check that the rest of the cycle gives an alternate route.

Many graph-induction errors happen because the smaller graph is smaller but no longer belongs to the class named in the inductive hypothesis.

## Common Smaller Graphs

> **Pattern.** For degree-counting theorems, delete a vertex and all incident edges.

This is the move in the degree-sum theorem.

> **Pattern.** For tree-counting theorems, delete a leaf and its unique incident edge.

This is the move in the proof that nonempty trees satisfy $|V(T)|=|E(T)|+1$.

> **Pattern.** For spanning-tree theorems, delete an edge from a cycle.

This keeps the graph connected and keeps the vertex set unchanged.

## Core Summary

Induction on graphs is induction on order or size. A complete proof must define the smaller graph, prove it still satisfies the inductive hypothesis, and then translate the conclusion back to the original graph. The graph operation is the mathematical content of the induction step.

Before truth values enter the discussion, the course needs a smaller distinction.

Logic does not study every piece of language. It studies sentences that make definite claims and can therefore be evaluated.

## Declarative Sentences

The basic objects in this unit are declarative sentences.

A declarative sentence presents the world as being some way. Because it makes an assertion, it is the kind of thing that can be correct or incorrect.

By contrast, the following expressions are meaningful without being truth-valued:

- "Close the door."
- "Did you close the door?"
- "The door."

The first is a command. The second is a question. The third is a fragment. None of them asserts a state of affairs, so none of them is the kind of thing logic will evaluate as true or false.

Now compare them with "The door is closed."

That sentence does assert a state of affairs. It says that the door has a particular property. This makes it a candidate for truth or falsity.

## Sentences and Objects

This distinction matters immediately because the course will keep separating a sentence from the object the sentence is about.

Consider "$7$ is prime."

Here the object is $7$.

The sentence is the full assertion "$7$ is prime."

The truth value belongs to the sentence, not to the number. The number is an object of discourse. The sentence is a claim about that object.

Later, when logical connectives are introduced, this distinction becomes operational. The connective does not act on objects like $7$. It acts on sentences that already carry logical content.

## Criteria for Sentencehood

When an expression appears in the notes or on a problem set, three questions are worth checking:

1. Is the expression an assertion rather than a command, question, or fragment?
2. Does it describe the world as being some way?
3. Could that description succeed or fail?

If the answer to the third question is no, then the expression is not the right kind of object for the present unit.

## Truth Value as a Type of Evaluation

The phrase "has a truth value" should be read carefully.

It does not mean the expression has already been shown true or false. It means the expression is the right kind of object for that evaluation.

This is the first gate in the unit. The next note studies what it means for a declarative sentence to receive the verdict true or false.

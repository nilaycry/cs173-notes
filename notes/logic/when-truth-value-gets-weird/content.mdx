The early logic notes begin with stable cases for a reason. Once that baseline is in place, the course can show where the naive picture stops working smoothly.

## The Naive Picture

A first model of truth value says:

- every declarative sentence has a truth value
- the only task is to determine which one

That model works well for many mathematical sentences.

It does not work equally well for every self-referential example.

## Why These Examples Appear

The purpose of these examples is not to turn the unit into a philosophy seminar.

Their purpose is to show that truth evaluation has assumptions underneath it. The course wants students to stop treating sentencehood and truth value as automatic just because an expression resembles ordinary English.

## The Right Lesson to Take

The main lesson is not that logic is unstable.

The main lesson is that logic is careful about what counts as a sentence, what kind of sentence is being evaluated, and what framework the evaluation is happening inside.

That care is one of the habits the rest of the course keeps asking for.

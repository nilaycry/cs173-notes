Countability asks whether a set can be organized by natural numbers. At this point the word "count" changes its flavor: we are not trying to produce a finite answer, but to design a precise encoding.

## Countable Sets

A set $X$ is **countable** if

$$
|X|\leq |\mathbb{N}|.
$$

Equivalently, $X$ is countable if there exists an injection

$$
f:X\to\mathbb{N}.
$$

Finite sets are countable. Some infinite sets are also countable.

## Countably Infinite

A set is **countably infinite** when it is both countable and infinite.

The standard examples are

$$
\mathbb{N},\quad \mathbb{Z},\quad \mathbb{N}\times\mathbb{N},\quad \mathbb{Q}.
$$

## Finite Strings

For an alphabet $A$, a finite string over $A$ of length $n\in\mathbb{N}$ is a function

$$
s:n\to A.
$$

If $A$ is finite, then for each fixed $n$, there are finitely many strings of length $n$ over $A$.

The set of all finite strings over a finite alphabet is countable: group the strings by length, and list each finite group before moving to the next length.

## Infinite Strings

An infinite string over $A$ is a function

$$
s:\mathbb{N}\to A.
$$

An infinite binary string is a function

$$
s:\mathbb{N}\to\{0,1\}.
$$

## One Contiguous Block of Ones

PS08 asks for the size of the set of infinite binary strings that contain one contiguous finite substring consisting only of $1$s and have zeros everywhere else. The trick is not to stare at the whole infinite string. Find the finite data that determines it.

A string of this form is determined by two natural numbers:

- the starting position $a\in\mathbb{N}$;
- the positive length $\ell\in\mathbb{N}^+$ of the block of $1$s.

Given $(a,\ell)$, define $s_{a,\ell}:\mathbb{N}\to\{0,1\}$ by

$$
s_{a,\ell}(i)\coloneqq
\begin{cases}
1, & a\leq i<a+\ell,\\
0, & \text{otherwise}.
\end{cases}
$$

Thus the set of such strings injects into

$$
\mathbb{N}\times\mathbb{N}^+.
$$

Since $\mathbb{N}\times\mathbb{N}^+$ is countable, the set of such strings is countable.

It is also infinite: for each $a\in\mathbb{N}$, the string with a single $1$ at position $a$ is different. Therefore the set is countably infinite.

## Core Summary

Countability problems are coding problems. To prove a set is countable, encode each object using natural-number data. A single finite block inside an infinite binary string is manageable because the whole string is determined by two numbers: where the block begins and how long it lasts.

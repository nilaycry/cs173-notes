Many pigeonhole proofs fail because the first codomain is too large. The handshake-count theorem shows the repair: use the actual set of possible outputs, then prove that this set is smaller than the domain.

## The Handshake-Count Theorem

> **Theorem.** Let $n\in\mathbb{N}$ with $n\geq 2$. In any group of $n$ people, there exist two distinct people who have shaken the same number of other people's hands.

This theorem is not proved by saying only that there are $n$ people and $n$ possible counts. The possible counts first appear to be too many.

## The Model

Let $A$ be the set of people, with $|A|=n$.

For each $p\in A$, let $d(p)$ be the number of other people whose hands $p$ has shaken. Then

$$
d:A\to\mathbb{N}
$$

records the handshake count of each person.

Since no one shakes their own hand,

$$
0\leq d(p)\leq n-1.
$$

The broad set of possible counts is

$$
D\coloneqq \{\,r\in\mathbb{N}\mid r\leq n-1\,\}.
$$

But $|D|=n$, so the function $d:A\to D$ does not force a collision.

## The Missing Extreme

> **Observation.** The counts $0$ and $n-1$ cannot both occur.

If someone has count $n-1$, then that person shook hands with everyone else. Therefore every other person has shaken at least one hand, so no one has count $0$.

If someone has count $0$, then that person shook hands with no one. Therefore no person shook hands with everyone else, so no one has count $n-1$.

## The Smaller Codomain

Let

$$
B\coloneqq \{\,d(p)\mid p\in A\,\}
$$

be the set of actual handshake counts.

Because the two extreme values cannot both appear, $B$ is missing at least one value from $D$. Therefore

$$
|B|\leq n-1.
$$

Now regard the same rule as a function

$$
d:A\to B.
$$

Since $|A|=n>|B|$, the pigeonhole principle says $d$ is not injective. Hence there exist distinct people $p,q\in A$ such that

$$
d(p)=d(q).
$$

Thus two distinct people have shaken the same number of hands.

## Graph Translation

People can be treated as vertices, and handshakes can be treated as edges. The theorem becomes:

> **Theorem.** If $G$ is a finite simple graph with at least two vertices, then there exist distinct vertices $u,v\in V(G)$ such that
>
> $$
> \deg_G(u)=\deg_G(v).
> $$

The proof is the same: degree values lie between $0$ and $n-1$, but the values $0$ and $n-1$ cannot both occur.

## Core Summary

The main move is to shrink the codomain. First define the natural output function. Then identify which output values can actually occur. Once the actual codomain is smaller than the domain, pigeonhole gives the collision.

# Course Outline

This is a starter outline for the `CS 173` site. It should be refined once the
source material is in place.

## Throughline

The course is not just a sequence of topics. It is also a sequence of changes in
how mathematical writing works.

The site should make both visible:

- what the content is
- what a proof in this part of the course is supposed to look like

## Suggested high-level structure

### Foundations

- what this course is really asking you to do
- how to read a mathematical statement
- how proof style changes across the course

### Logic

- propositions and open sentences
- connectives and truth tables
- negation
- quantified statements
- logical equivalence
- formal derivations if needed by the course

### Proof Methods

- direct proof
- contrapositive
- contradiction
- existence and uniqueness language if relevant

### Sets

- elements versus subsets
- set operations
- subset proofs
- proving set identities

### Functions and Relations

- functions, domain, codomain, image
- injective, surjective, bijective
- composition and inverses
- relations and equivalence relations

### Induction and Recursion

- induction structure
- strong induction if covered
- recursive definitions and proof use

### Counting

- sum and product rules
- permutations and combinations
- counting arguments
- case splits and bijective counting if covered

### Graph Theory

- graph definitions
- paths, cycles, connectivity
- trees
- structural graph proofs

## Companion note types

Alongside content notes, this project should also include:

- unit writing guides
- common mistake notes
- problem sets
- exam review pages

## Status

This is only the starting structure. It should be updated once the lecture notes,
problem sets, and practice midterms are in the repository.

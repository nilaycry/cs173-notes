# Source Materials

Put course materials here.

## Folder guide

- `past-lecture-notes/`
  - older full lecture note packets
- `past-lecture-notes/primary/`
  - the single most important master packet if there is one
- `current-course-notes/`
  - smaller notes released during the current semester
- `problem-sets/`
  - homework, worksheets, or discussion sheets
- `practice-midterms/`
  - practice exams and review sheets
- `proofs-and-solutions/`
  - solution keys, worked proofs, instructor examples
- `syllabus/`
  - syllabus, schedule, or topic list

## File tips

- PDFs are good.
- Keep original filenames when possible.
- If a file is messy, prefix it with a week or unit number.
- If one file is the main course backbone, place a copy in `past-lecture-notes/primary/`.
